Basics Of The Internet

  • The internet is a network of connected devices allowing us to communicate.
  • URL: Uniform Resource Locators. Identify the location of a site.
  • HTTP: Hypertext Transfer Protocol - standardized set of rules to communicate.
  • Server: Machine to handle requests from a client (The one who requests from the server.)
  • So What Happens?: Browser renders a response from the server. Turns code (HTML, CSS, JS etc.) into human readable content.

Front-End VS Back-End

  • Back End: The server side which constructs the response. Focused on languages and things that run on server.
  • Front End: The client side where the code is made visual/human readable. Focuses on HTML, CSS, JS - things that run on browser.

HTML, CSS, JavaScript

  • HTML: The structured content of the page.
  • CSS: The appearance of HTML on the page.
  • JavaScript: The actions on the page.
