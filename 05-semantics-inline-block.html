HTML Semantics

  • article - specifies independent, self-contained content.
  • section - defines section in a document.
  • header - represents a container for introductory content.
  • footer - a footer for a document or section.
  • nav - a set of navigation links.
  • aside - content aside from the content it is placed in (like a sidebar).
  • figure - tag specifies self-contained content, like illustrations, diagrams, photos, code listings, etc. figcaption - defines a caption for figure element.
  • main - Specifies the main content of a document
  • mark - Defines marked/highlighted text
  • time - Date/Time

Inline VS Block

Block level elements exist on their own line while inline elements are pushed together on the same line.

A "div" is a generic block level element to hold/group content. A "span" is an inline version of a div.
